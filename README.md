# Firefox OS DEVELOPERIFY

> Put your Firefox OS device into super developer mode

**WARNING: THIS CAN BRICK YOUR DEVICE**

*USE WITH CAUTION*

## How to use

Clone, then cd to the dir and run

`
npm install
`

then just run

`
node index.js
`
